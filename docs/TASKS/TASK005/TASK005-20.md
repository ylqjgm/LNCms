### 2. 实现Logout方法

```go
// Logout 用户登出
// @Summary 用户登出
// @Description 用户登出接口，将token加入黑名单
// @Tags 用户管理
// @Accept json
// @Produce json
// @Security BearerAuth
// @Success 200 {object} response.Envelope
// @Failure 400 {object} response.Envelope
// @Failure 401 {object} response.Envelope
// @Failure 500 {object} response.Envelope
// @Router /api/v1/users/logout [post]
func (h *userHandler) Logout(c *gin.Context) {
    // 1. 从请求头获取JWT token
    token := extractTokenFromHeader(c)
    if token == "" {
        response.BadRequest(c, "请求参数错误", errors.New("缺少认证token"))
        return
    }
    
    // 2. 调用AuthService.Logout
    err := h.authService.Logout(c.Request.Context(), token)
    if err != nil {
        if isUnauthorizedError(err) {
            response.Unauthorized(c, "认证失败", err)
            return
        }
        response.InternalServerError(c, "登出失败", err)
        return
    }
    
    // 3. 格式化响应（Envelope格式）
    response.OK(c, nil, "登出成功")
}
```

### 3. 实现Token提取辅助函数

```go
// extractTokenFromHeader 从请求头提取JWT token
// 支持格式：Authorization: Bearer {token}
func extractTokenFromHeader(c *gin.Context) string {
    authHeader := c.GetHeader("Authorization")
    if authHeader == "" {
        return ""
    }
    
    // 检查Bearer前缀
    const bearerPrefix = "Bearer "
    if len(authHeader) < len(bearerPrefix) {
        return ""
    }
    
    if authHeader[:len(bearerPrefix)] != bearerPrefix {
        return ""
    }
    
    // 提取token
    token := authHeader[len(bearerPrefix):]
    if token == "" {
        return ""
    }
    
    return token
}
```

### 5. 注意事项

- **Token提取**：从Authorization请求头提取Bearer token
- **Token格式**：支持标准格式 `Authorization: Bearer {token}`
- **错误处理**：如果token缺失或格式错误，返回400 Bad Request
- **响应格式**：所有响应使用Envelope格式
- **状态码**：登出成功返回200 OK
- **安全性**：登出时将token加入黑名单，防止token被继续使用
- **认证要求**：登出操作需要认证，在路由配置中应用JWT认证中间件
