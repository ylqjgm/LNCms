## 实现指南

### 1. 设计AuthorSession表结构

AuthorSession表用于存储JWT token信息，支持token黑名单管理。表结构设计如下：

- **id**: BIGSERIAL主键
- **author_id**: BIGINT，外键关联authors表
- **token**: TEXT，存储JWT token字符串
- **token_type**: VARCHAR(20)，token类型（如：access、refresh）
- **expires_at**: TIMESTAMPTZ，token过期时间
- **is_blacklisted**: BOOLEAN，是否在黑名单中
- **blacklisted_at**: TIMESTAMPTZ，加入黑名单的时间
- **created_at**: TIMESTAMPTZ，创建时间
- **updated_at**: TIMESTAMPTZ，更新时间

### 2. 创建AuthorSession模型文件

在 `internal/models/` 目录下创建 `author_session.go` 文件，定义AuthorSession模型：

```go
package models

import (
    "time"
    "gorm.io/gorm"
)

// TokenType token类型常量
const (
    TokenTypeAccess  = "access"  // 访问token
    TokenTypeRefresh = "refresh" // 刷新token
)

// AuthorSession 作者会话模型
// 用于存储JWT token信息，支持token黑名单管理
type AuthorSession struct {
    ID            int64      `gorm:"primaryKey;column:id;type:BIGSERIAL" json:"id"`
    AuthorID      int64      `gorm:"column:author_id;type:BIGINT;not null;index:idx_author_sessions_author_id" json:"author_id"`
    Token         string     `gorm:"column:token;type:TEXT;not null;uniqueIndex:uk_author_sessions_token" json:"-"`
    TokenType     string     `gorm:"column:token_type;type:VARCHAR(20);not null;default:'access';index:idx_author_sessions_token_type" json:"token_type"`
    ExpiresAt     time.Time  `gorm:"column:expires_at;type:TIMESTAMPTZ;not null;index:idx_author_sessions_expires_at" json:"expires_at"`
    IsBlacklisted bool       `gorm:"column:is_blacklisted;type:BOOLEAN;default:false;index:idx_author_sessions_is_blacklisted" json:"is_blacklisted"`
    BlacklistedAt *time.Time `gorm:"column:blacklisted_at;type:TIMESTAMPTZ" json:"blacklisted_at,omitempty"`
    CreatedAt     time.Time  `gorm:"column:created_at;type:TIMESTAMPTZ;default:CURRENT_TIMESTAMP" json:"created_at"`
    UpdatedAt     time.Time  `gorm:"column:updated_at;type:TIMESTAMPTZ;default:CURRENT_TIMESTAMP" json:"updated_at"`
    
    // 关联关系
    Author *Author `gorm:"foreignKey:AuthorID;references:ID" json:"author,omitempty"`
}

// TableName 指定表名
func (AuthorSession) TableName() string {
    return "author_sessions"
}

// BeforeCreate 创建前钩子
func (as *AuthorSession) BeforeCreate(tx *gorm.DB) error {
    now := time.Now()
    if as.CreatedAt.IsZero() {
        as.CreatedAt = now
    }
    if as.UpdatedAt.IsZero() {
        as.UpdatedAt = now
    }
    return nil
}

// BeforeUpdate 更新前钩子
func (as *AuthorSession) BeforeUpdate(tx *gorm.DB) error {
    as.UpdatedAt = time.Now()
    return nil
}
```

### 3. 添加模型注释

在模型文件中添加详细的中文注释：

```go
// AuthorSession 作者会话模型
// 用于存储JWT token信息，支持token黑名单管理
// 当作者登录时创建会话记录，登出时将token加入黑名单
type AuthorSession struct {
    // ID 会话ID，自增主键
    ID int64 `gorm:"primaryKey;column:id;type:BIGSERIAL" json:"id"`
    
    // AuthorID 作者ID，外键关联authors表
    AuthorID int64 `gorm:"column:author_id;type:BIGINT;not null;index:idx_author_sessions_author_id" json:"author_id"`
    
    // Token JWT token字符串，唯一标识符
    Token string `gorm:"column:token;type:TEXT;not null;uniqueIndex:uk_author_sessions_token" json:"-"`
    
    // TokenType token类型：access-访问token，refresh-刷新token
    TokenType string `gorm:"column:token_type;type:VARCHAR(20);not null;default:'access';index:idx_author_sessions_token_type" json:"token_type"`
    
    // ExpiresAt token过期时间
    ExpiresAt time.Time `gorm:"column:expires_at;type:TIMESTAMPTZ;not null;index:idx_author_sessions_expires_at" json:"expires_at"`
    
    // IsBlacklisted 是否在黑名单中，TRUE-已加入黑名单，FALSE-未加入黑名单
    IsBlacklisted bool `gorm:"column:is_blacklisted;type:BOOLEAN;default:false;index:idx_author_sessions_is_blacklisted" json:"is_blacklisted"`
    
    // BlacklistedAt 加入黑名单的时间
    BlacklistedAt *time.Time `gorm:"column:blacklisted_at;type:TIMESTAMPTZ" json:"blacklisted_at,omitempty"`
    
    // CreatedAt 创建时间，自动设置为当前时间
    CreatedAt time.Time `gorm:"column:created_at;type:TIMESTAMPTZ;default:CURRENT_TIMESTAMP" json:"created_at"`
    
    // UpdatedAt 更新时间，每次更新时自动更新
    UpdatedAt time.Time `gorm:"column:updated_at;type:TIMESTAMPTZ;default:CURRENT_TIMESTAMP" json:"updated_at"`
    
    // 关联关系
    Author *Author `gorm:"foreignKey:AuthorID;references:ID" json:"author,omitempty"`
}
```

### 4. 定义模型方法

为AuthorSession模型添加常用的辅助方法：

```go
// IsExpired 检查token是否已过期
func (as *AuthorSession) IsExpired() bool {
    return time.Now().After(as.ExpiresAt)
}

// IsValid 检查token是否有效（未过期且未加入黑名单）
func (as *AuthorSession) IsValid() bool {
    return !as.IsExpired() && !as.IsBlacklisted
}

// Blacklist 将token加入黑名单
func (as *AuthorSession) Blacklist() {
    now := time.Now()
    as.IsBlacklisted = true
    as.BlacklistedAt = &now
}

// Unblacklist 将token从黑名单移除
func (as *AuthorSession) Unblacklist() {
    as.IsBlacklisted = false
    as.BlacklistedAt = nil
}
```

### 5. 设计索引

- **主键索引**：`pk_author_sessions` (id) - 自动创建
- **唯一索引**：`uk_author_sessions_token` (token) - 确保token唯一
- **普通索引**：`idx_author_sessions_author_id` (author_id) - 用于查询作者的会话
- **普通索引**：`idx_author_sessions_token_type` (token_type) - 用于按类型查询
- **普通索引**：`idx_author_sessions_expires_at` (expires_at) - 用于查询过期token
- **普通索引**：`idx_author_sessions_is_blacklisted` (is_blacklisted) - 用于查询黑名单token

### 6. 注意事项

- **Token字段**：使用TEXT类型存储JWT token，因为token可能较长
- **唯一索引**：token字段必须唯一，确保每个token只对应一个会话
- **黑名单机制**：使用is_blacklisted字段标记黑名单，支持快速查询
- **过期时间**：expires_at字段用于判断token是否过期，支持自动清理过期token
- **JSON标签**：Token字段使用 `json:"-"` 避免序列化到JSON响应中
- **外键关联**：AuthorID关联authors表，支持级联删除（可选）
