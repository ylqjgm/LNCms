### 2. 设计小说分卷表结构

设计 `novel_volumes` 表，包含小说分卷基本信息：

```sql
CREATE TABLE novel_volumes (
    id BIGSERIAL PRIMARY KEY,
    novel_id BIGINT NOT NULL,
    volume_name VARCHAR(200) NOT NULL,
    volume_description TEXT,
    volume_order INTEGER NOT NULL DEFAULT 0,
    is_published BOOLEAN DEFAULT FALSE,
    published_at TIMESTAMPTZ,
    created_at TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP,
    CONSTRAINT fk_novel_volumes_novel_id FOREIGN KEY (novel_id) REFERENCES novels(id) ON DELETE CASCADE,
    CONSTRAINT uk_novel_volumes_novel_id_volume_order UNIQUE (novel_id, volume_order)
);

-- 创建索引
CREATE INDEX idx_novel_volumes_novel_id ON novel_volumes(novel_id);
CREATE INDEX idx_novel_volumes_volume_order ON novel_volumes(volume_order);
CREATE INDEX idx_novel_volumes_is_published ON novel_volumes(is_published);
CREATE UNIQUE INDEX uk_novel_volumes_novel_id_volume_order ON novel_volumes(novel_id, volume_order);

-- 创建触发器自动更新 updated_at
CREATE OR REPLACE FUNCTION update_novel_volumes_updated_at()
RETURNS TRIGGER AS $$
BEGIN
    NEW.updated_at = CURRENT_TIMESTAMP;
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER update_novel_volumes_updated_at
    BEFORE UPDATE ON novel_volumes
    FOR EACH ROW
    EXECUTE FUNCTION update_novel_volumes_updated_at();

-- 添加表注释
COMMENT ON TABLE novel_volumes IS '小说分卷表，存储小说分卷信息';
COMMENT ON COLUMN novel_volumes.id IS '分卷ID，自增主键';
COMMENT ON COLUMN novel_volumes.novel_id IS '小说ID，外键关联novels表';
COMMENT ON COLUMN novel_volumes.volume_name IS '分卷名称';
COMMENT ON COLUMN novel_volumes.volume_description IS '分卷描述';
COMMENT ON COLUMN novel_volumes.volume_order IS '分卷排序，数字越小越靠前，用于排序显示';
COMMENT ON COLUMN novel_volumes.is_published IS '是否发布，TRUE-已发布，FALSE-未发布';
COMMENT ON COLUMN novel_volumes.published_at IS '发布时间';
COMMENT ON COLUMN novel_volumes.created_at IS '创建时间，自动设置为当前时间';
COMMENT ON COLUMN novel_volumes.updated_at IS '更新时间，每次更新时自动更新';
```

### 3. 设计约束

- **主键约束**：`pk_novel_volumes` (id)
- **外键约束**：`fk_novel_volumes_novel_id` (novel_id) 关联 novels(id)，级联删除
- **唯一约束**：`uk_novel_volumes_novel_id_volume_order` (novel_id, volume_order) - 确保同一小说的分卷排序唯一
- **非空约束**：novel_id、volume_name 必须非空
- **默认值**：volume_order 默认为 0，is_published 默认为 FALSE

### 4. 设计索引

- **主键索引**：`pk_novel_volumes` (id) - 自动创建
- **外键索引**：`idx_novel_volumes_novel_id` (novel_id) - 用于查询小说的分卷
- **普通索引**：`idx_novel_volumes_volume_order` (volume_order) - 用于排序查询
- **普通索引**：`idx_novel_volumes_is_published` (is_published) - 用于发布状态筛选
- **唯一索引**：`uk_novel_volumes_novel_id_volume_order` (novel_id, volume_order) - 确保唯一性

### 5. 考虑性能优化

- 为 novel_id 创建索引，支持快速查询小说的分卷
- 为 volume_order 创建索引，支持排序查询
- 使用复合唯一索引，确保同一小说的分卷排序唯一
- 使用触发器自动更新 updated_at 字段
