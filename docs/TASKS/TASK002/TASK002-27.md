### 2. 设计音频分季表结构

设计 `audio_seasons` 表，包含音频分季基本信息：

```sql
CREATE TABLE audio_seasons (
    id BIGSERIAL PRIMARY KEY,
    audio_id BIGINT NOT NULL,
    season_name VARCHAR(200) NOT NULL,
    season_description TEXT,
    season_order INTEGER NOT NULL DEFAULT 0,
    is_published BOOLEAN DEFAULT FALSE,
    published_at TIMESTAMPTZ,
    created_at TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP,
    CONSTRAINT fk_audio_seasons_audio_id FOREIGN KEY (audio_id) REFERENCES audios(id) ON DELETE CASCADE,
    CONSTRAINT uk_audio_seasons_audio_id_season_order UNIQUE (audio_id, season_order)
);

-- 创建索引
CREATE INDEX idx_audio_seasons_audio_id ON audio_seasons(audio_id);
CREATE INDEX idx_audio_seasons_season_order ON audio_seasons(season_order);
CREATE INDEX idx_audio_seasons_is_published ON audio_seasons(is_published);
CREATE UNIQUE INDEX uk_audio_seasons_audio_id_season_order ON audio_seasons(audio_id, season_order);

-- 创建触发器自动更新 updated_at
CREATE OR REPLACE FUNCTION update_audio_seasons_updated_at()
RETURNS TRIGGER AS $$
BEGIN
    NEW.updated_at = CURRENT_TIMESTAMP;
    RETURN NEW;
END;
$$ LANGUAGE plpgsql;

CREATE TRIGGER update_audio_seasons_updated_at
    BEFORE UPDATE ON audio_seasons
    FOR EACH ROW
    EXECUTE FUNCTION update_audio_seasons_updated_at();

-- 添加表注释
COMMENT ON TABLE audio_seasons IS '音频分季表，存储音频分季信息';
COMMENT ON COLUMN audio_seasons.id IS '分季ID，自增主键';
COMMENT ON COLUMN audio_seasons.audio_id IS '音频ID，外键关联audios表';
COMMENT ON COLUMN audio_seasons.season_name IS '分季名称';
COMMENT ON COLUMN audio_seasons.season_description IS '分季描述';
COMMENT ON COLUMN audio_seasons.season_order IS '分季排序，数字越小越靠前，用于排序显示';
COMMENT ON COLUMN audio_seasons.is_published IS '是否发布，TRUE-已发布，FALSE-未发布';
COMMENT ON COLUMN audio_seasons.published_at IS '发布时间';
COMMENT ON COLUMN audio_seasons.created_at IS '创建时间，自动设置为当前时间';
COMMENT ON COLUMN audio_seasons.updated_at IS '更新时间，每次更新时自动更新';
```

### 3. 设计约束

- **主键约束**：`pk_audio_seasons` (id)
- **外键约束**：`fk_audio_seasons_audio_id` (audio_id) 关联 audios(id)，级联删除
- **唯一约束**：`uk_audio_seasons_audio_id_season_order` (audio_id, season_order) - 确保同一音频的分季排序唯一
- **非空约束**：audio_id、season_name 必须非空
- **默认值**：season_order 默认为 0，is_published 默认为 FALSE

### 4. 设计索引

- **主键索引**：`pk_audio_seasons` (id) - 自动创建
- **外键索引**：`idx_audio_seasons_audio_id` (audio_id) - 用于查询音频的分季
- **普通索引**：`idx_audio_seasons_season_order` (season_order) - 用于排序查询
- **普通索引**：`idx_audio_seasons_is_published` (is_published) - 用于发布状态筛选
- **唯一索引**：`uk_audio_seasons_audio_id_season_order` (audio_id, season_order) - 确保唯一性

### 5. 考虑性能优化

- 为 audio_id 创建索引，支持快速查询音频的分季
- 为 season_order 创建索引，支持排序查询
- 使用复合唯一索引，确保同一音频的分季排序唯一
- 使用触发器自动更新 updated_at 字段
