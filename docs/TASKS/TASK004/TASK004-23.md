### 2. 创建AuthorType模型

在 `internal/pkg/model/author_type.go` 文件中创建AuthorType结构体：

```go
package model

import (
    "time"
    "gorm.io/gorm"
)

// AuthorType 作者类型模型
// 用于存储作者类型信息，如原创、转载、翻译等
// 表名: author_types
// 索引: pk_author_types (id), uk_author_types_code (code), idx_author_types_sort_order (sort_order)
type AuthorType struct {
    // ID 作者类型ID，自增主键
    ID int64 `gorm:"primaryKey;autoIncrement" json:"id"`
    
    // Code 作者类型代码，唯一标识符，用于系统内部识别
    Code string `gorm:"type:varchar(50);not null;uniqueIndex:uk_author_types_code" json:"code"`
    
    // Name 作者类型名称，用于显示
    Name string `gorm:"type:varchar(100);not null" json:"name"`
    
    // Description 作者类型描述，说明作者类型的用途和特点
    Description *string `gorm:"type:text" json:"description,omitempty"`
    
    // SortOrder 显示顺序，用于控制作者类型在列表中的显示顺序
    SortOrder int `gorm:"type:integer;default:0;index:idx_author_types_sort_order" json:"sort_order"`
    
    // CreatedAt 创建时间，自动设置为当前时间
    CreatedAt time.Time `gorm:"type:timestamptz;default:current_timestamp" json:"created_at"`
    
    // UpdatedAt 更新时间，自动设置为当前时间
    UpdatedAt time.Time `gorm:"type:timestamptz;default:current_timestamp" json:"updated_at"`
    
    // DeletedAt 软删除时间（GORM软删除支持）
    DeletedAt gorm.DeletedAt `gorm:"index" json:"deleted_at,omitempty"`
}

// TableName 指定表名
func (AuthorType) TableName() string {
    return "author_types"
}
```

### 3. 验证模型定义

确保模型定义符合以下要求：

- 主键字段使用 `bigserial` 类型（GORM会自动映射为int64）
- `code` 字段使用 `varchar(50)` 类型，设置唯一索引
- `name` 字段使用 `varchar(100)` 类型
- `description` 字段使用 `text` 类型，允许为空
- `sort_order` 字段使用 `integer` 类型，设置默认值和索引
- 时间字段使用 `timestamptz` 类型，支持时区
- 所有字段都有适当的GORM标签和JSON标签
- 模型实现了 `TableName()` 方法，明确指定表名
- 支持GORM软删除功能（DeletedAt字段）
