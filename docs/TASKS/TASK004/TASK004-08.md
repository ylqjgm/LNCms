### 2. 创建UserGroup模型

在 `internal/pkg/model/user_group.go` 文件中创建UserGroup结构体：

```go
package model

import (
    "time"
    "gorm.io/gorm"
)

// UserGroup 用户组模型
// 用于存储网站用户组信息，用于对用户进行分组管理，支持权限分配
// 表名: user_groups
// 索引: pk_user_groups (id), uk_user_groups_group_name (group_name), idx_user_groups_is_active (is_active)
type UserGroup struct {
    // ID 用户组ID，自增主键
    ID int64 `gorm:"primaryKey;autoIncrement" json:"id"`
    
    // GroupName 用户组名称，唯一标识符
    GroupName string `gorm:"type:varchar(100);not null;uniqueIndex:uk_user_groups_group_name" json:"group_name"`
    
    // GroupDescription 用户组描述，说明用户组的用途和权限范围
    GroupDescription *string `gorm:"type:text" json:"group_description,omitempty"`
    
    // IsActive 是否启用，TRUE-启用，FALSE-禁用
    IsActive bool `gorm:"type:boolean;default:true;index:idx_user_groups_is_active" json:"is_active"`
    
    // CreatedAt 创建时间，自动设置为当前时间
    CreatedAt time.Time `gorm:"type:timestamptz;default:current_timestamp" json:"created_at"`
    
    // UpdatedAt 更新时间，自动设置为当前时间
    UpdatedAt time.Time `gorm:"type:timestamptz;default:current_timestamp" json:"updated_at"`
    
    // DeletedAt 软删除时间（GORM软删除支持）
    DeletedAt gorm.DeletedAt `gorm:"index" json:"deleted_at,omitempty"`
}

// TableName 指定表名
func (UserGroup) TableName() string {
    return "user_groups"
}
```

- `group_description` 字段使用 `text` 类型，允许为空
- `is_active` 字段使用 `boolean` 类型，设置默认值和索引
- 时间字段使用 `timestamptz` 类型，支持时区
- 所有字段都有适当的GORM标签和JSON标签
- 模型实现了 `TableName()` 方法，明确指定表名
- 支持GORM软删除功能（DeletedAt字段）
