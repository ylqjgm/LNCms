## 模块概述

**Web** 为程序前端模块，负责对核心模块 API 进行数据展示、提交请求。

## 开发技术

- **开发语言**: JavaScript ES6
- **包管理**: Bun
- **构建工具**: Vite
- **开发框架**: Vue3
- **UI组件**: Element Plus
- **状态管理**: Pinia
- **状态持久化**: IndexedDB（使用 Pinia 插件实现状态持久化）
- **路由**: Vue Router
- **HTTP 客户端**: Fetch API
- **CSS 预处理器**: SCSS

### API 接口

- 使用核心模块提供的 RESTful HTTP API、WebScoket 接口服务
- 所有 API 遵循 OpenAPI 3.1.1 规范
- 所有响应统一遵循 Envelope 响应格式
- 核心模块通过 RESTful HTTP API、WebScoket 接口与前端模块进行通信
