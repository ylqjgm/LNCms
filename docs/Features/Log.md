## 模块概述

**Log** 为程序日志模块，负责提供项目日志记录、查询、分析功能。

## 开发技术

- **开发语言**: Golang 1.24+
- **开发框架**: Gin
- **数据库**: PostgreSQL
- **缓存**: Redis

### gRPC 通信

- 对外提供 gRPC 接口服务
- 核心模块、存储模块、采集模块通过 gRPC 接口与日志模块进行通信
- 模块间使用 gRPC 均支持双向通信

## 功能描述

### 1. 日志存储

#### 1.1 日志写入

各模块写入统一日志信息至日志模块。

#### 1.2 日志归档

日志模块根据日志配置要求，自动将过期日志进行归档处理。

### 2. 日志查询

日志模块提供核心模块进行日志查询功能。

- 日志列表
- 分类日志
- 日志筛选
- 日志查询
- 日志导出

### 3. 日志统计

日志模块提供核心模块进行日志统计功能。

- 日志统计
- 日志分析

### 5. 健康检查

提供核心模块健康检查接口。

- 运行状态
- 连接数值
